# SilentGuardian
SilentGuardian is a C# application designed to monitor user activity on a Windows PC and trigger alerts based on configurable behavioral conditions. It communicates with a Telegram bot for remote control and notifications.

## Key Features
- Global mouse and keyboard activity hooks
Activity analysis and behavior monitoring
Locks the workstation (Autolock on alert is too available)
Captures screenshots
Records screen video
Status monitoring via Telegram bot
Behavior and thresholds fully configurable through config.json
